HOW TO RUN OUR PROGRAM?:
1. First, open terminal and navigate to submission directory.
2. Run our bash script by typing 'bash flightScanner.sh <arg1> <arg2> <...>' in terminal.
3. Run our makefile by typing 'make' in terminal.
4. Run the following command in terminal: 'export LD_LIBRARY_PATH=./build'.
5. Run your wanted program:
	5.1. For program#1, run './arrivals.out <arg1> <arg2> <...>' in  terminal.
	5.2. For program#2, run './full_schedule.out <arg>' in terminal (Can also get multiple arguments).
	5.3. For program#3, run './airplane.out <arg1> <arg2> <...>' in terminal (Gets icao24's).
	5.4. For program#4, run './RefreshDB.out' in terminal.
6. To clean up, type 'make clean' in terminal.
